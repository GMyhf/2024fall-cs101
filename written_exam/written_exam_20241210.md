2）写出程序的代码（3分）

```python
inp = input().split()
n = int(inp[0])
k = int(inp[1])

l = []
for i in range(n):
    l.append(int(input()))
max = 0
for i in range(n - k + 1):
    v = sum(l[i:i + k])
    if v > max:
        max = v

print(max)
```
